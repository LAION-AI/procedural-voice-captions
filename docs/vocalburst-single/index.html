VocalBurst — single-burst classifier (pure clips)

This is the specialised "single-burst" model. It is trained ONLY on clean clips that contain exactly one vocal burst and no speech, so it is sharper at naming the exact burst type. Same build as the main model: frozen VoiceCLAP ears → 768-number fingerprint → a small trained MLP (1024×3) → a probability per class (softmax, i.e. the probabilities sum to 1 — it picks the single most likely type). "no-burst gate": if P(no burst) ≥ 0.5 we declare 🔇 no burst and hide the rest.

On these held-out pure clips: exact top-1 60%, correct in top-3 88%; on the coarse 16 families ~0.87 mAP. 390 clips, ~5/class.
