Conclusions

  • more-synthetic labels are only ~52% reliable. Its synthetic generator produces the intended burst well for acoustically-distinct classes (Scream, Exasperated Sigh, Deep Breathing, playful whistling → ~100%) but frequently emits a generic breath/sigh for subtle or niche classes (Surprised Gasp, Snort, Gulps, Tsk, Wolf Whistle, Normal Breathing → ~0%). → re-label from audio.
  • gemini_pro_25 tags are ~75% reliable — much better (real-ish voice-acting audio, Gemini-2.5-Pro per-event tags), and it is genuinely multi-label (several burst classes per clip). Weakest on Finger Snaps, Shriek, Ahem, Snicker, Clears Throat.
  • Common failure = collapse to breathing/sigh: most confusions are niche→(Breathing/Gasp/Sigh), i.e. the synth couldn't render the fine burst. This is exactly the long-tail the classifier must not learn from noisy labels.
  • Recommendation: trust the confirmed (yes) subset directly; re-label the rest from audio (≤$20, see above); keep gemini_pro_25 as multi-label positives; use only the high-agreement classes from more-synthetic as-is.
