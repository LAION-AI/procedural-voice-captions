VocalBurst — two-stage detect + classify

The idea: instead of classifying the whole clip at once, first find WHERE a burst is, then name it.
1. Detector (laion/vocalburst-locator, a small model that scans the timeline) marks every moment whose "is this a non-speech burst?" score is ≥ 0.7 → candidate time segments.
2. For each candidate segment, the multi-label classifier (VoiceCLAP → MLP) names the burst via its top-1 prediction. If it isn't confident it's a real burst, the segment is discarded — no false alarm.

Green = the named burst matches a ground-truth burst in the clip. 40 speech-with-burst clips.
