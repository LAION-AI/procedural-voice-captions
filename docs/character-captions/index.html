Status: kept = confirmed, passed the duration gate & inserted inline; gated <Xs = a class was confirmed but the span was too short (global floor 0.30 s, 0.60 s for smack/click/slap groups); no_burst gate = rejected because p(no_burst) ≥ 0.5.

Gender gate: the dedicated Empathic-Insight gender expert (−2 masculine … +2 feminine) is used only as a confidence gate.
