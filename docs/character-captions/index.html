The archetype is passed to the LLM only if it names no gender (orc/ghost/dragon ok; man/woman/male/female suppressed) and gender is never guessed unless the annotations state it. Bursts highlighted.

The procedural SCRIPT now inserts the kept detected bursts inline (mapped to the sentence they occur in). Each card lists every detected burst span (locator@0.7 → multi-label confirm) with its duration. Spans shorter than 0.6 s are shown in red — very short detections are the ones that trigger hallucinated bursts. Status: kept = confirmed, passed the duration gate & inserted inline; gated <Xs = a class was confirmed but the span was too short (global floor 0.30 s, 0.60 s for smack/click/slap groups); no_burst gate = rejected because p(no_burst) ≥ 0.5.

Gender gate: the dedicated Empathic-Insight gender expert (−2 masculine … +2 feminine) is used only as a confidence gate. When |EI gender| is near zero the voice's gender is not clearly perceptible, so the caption omits gender entirely instead of asserting a possibly-wrong label; above the gate the well-calibrated VoiceNet label is kept. The clip's EI value (and whether gender was omitted) is shown next to each player.
