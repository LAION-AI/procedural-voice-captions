VocalBurst classifiers on MOSS character voices

Archetype voices generated by the MOSS-4.55B voice-acting model (zombie, ork, ranting, pain-scream, evil-ghost, mouse, dragon, fairy, goblin, ASMR…) — from the best-of-64 evolution grids — run through both vocal-burst classifiers. Each = VoiceCLAP embedding → a small MLP. The no-burst gate (P(no burst) ≥ 0.5 → 🔇) decides whether there's a burst at all.

Below: 10 clips where a burst WAS detected and 4 where none was (decided by the multi-label model's gate). We show what each model predicts, so you can listen and judge. There are no ground-truth labels here — this is the models applied to fresh, unlabelled character voices.
